# Voice Casting — Мия

Цель: отобрать голос Мии для фиксации в Suno Persona `MIYA_v1`. Сгенерить 18 версий по 3 направлениям (по 6 на вариант), отобрать шортлист, выбрать финал.

## Тестовая лирика

Фрагмент из "Первой тишины" — 1 куплет + 1 припев, ~45 сек. Показывает **оба режима**: интимный dry close-mic в куплете и soaring layered в припеве. Если эти 8 строк звучат правильно — звучит весь EP.

```
[Verse — Female voice, Miya, intimate dry close-mic, almost spoken over piano]
Канал один — молчит
Канал два — молчит
Канал три — несущая есть
Но в эфире никого

[Chorus — Female voice, Miya, soaring full voice with layered harmonies, distorted guitars enter]
Это первая тишина
Которая громче крика
Это первая тишина
Которая режет насквозь
```

## Процесс

1. **На каждый вариант — 6 генераций.** Custom Mode, Style = промпт варианта, лирика = тестовая (см. выше). Итого 18 версий.
2. **Слушать с наушниками**, не на колонках ноута.
3. **Записывать номера/ссылки понравившихся** в шортлист ниже.
4. После прослушки — **отдать мне шортлист (2–3 версии)**, выбираем финал и фиксируем Persona.

## Что слушать

- [ ] **Гендер** — Suno иногда переворачивает. Не тот пол → отбросить
- [ ] **Регистр** — mezzo/alto, не яркое сопрано
- [ ] **Куплет остаётся интимным** — не "разворачивается" в стадион
- [ ] **Припев реально открывается** — есть lift, не равный куплету
- [ ] **Эмоция читается** — личность, не генерик-сессионник
- [ ] **Стабильность** — нет autotune-warble, скачков питча, странного вибрато
- [ ] **Русский язык** — родной носитель, не американский акцент
- [ ] **Возраст** — 25-28, не подросток и не "ветеран"
