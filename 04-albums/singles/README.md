# HELIOTAIL — Singles

Отдельные треки вне альбомов: проверочные генерации, тематические meant-tests, soло-треки одного персонажа, B-сайды.

## Готовые

- [01-pervyy-zakat.md](./01-pervyy-zakat.md) — **"ПЕРВЫЙ ЗАКАТ"** ✅ Мия соло. Каноничный момент между РАЗРЫВом и СОРВУ — первый вечер одна на Хвост-9, тихая констатация до ярости. Intimate indie / ambient, ~3:00.
- [02-ehom.md](./02-ehom.md) — **"ЭХОМ"** ✅ Мия соло. Меланхолически-нежная медитация: что станет с ними после исчезновения. Адаптация структуры известной русской песни под HELIOTAIL universe. Intimate alt-electronic / ambient pop, ~3:30.

## Парковочные идеи

### "СЕМЬДЕСЯТ ДВА ЧАСА"

**Каноничный, Mia solo.** Альбом 2 timeline: окно в 72 часа между Программой ставящей условия (Эп.2 ПРЕДЛОЖЕНИЕ) и решением (Эп.3 ВЫБОР). Что у Мии в голове, пока NOX тоже **думает отдельно** (это первый раз, когда они сознательно не разделяют состояния через линк — каждый принимает решение сам).

- Звук: intimate indie с нарастающей тревогой, более тяжёлый чем "ПЕРВЫЙ ЗАКАТ"
- Драматургия: на пороге выбора, ещё может сказать "нет"
- Длина: ~3:30
- Якорная фраза-кандидат: *"Семьдесят два часа / Я считаю их одна"*
- Эмоциональный тон: не "перед сном" — для бодрого тестирования драматичного диапазона MIYA_v1
- **Расширяет нарратив Альбома 2**: заполняет gap между Эп.2 и Эп.3, может быть включён в reissue альбома как B-track

### "ГОЛОСОВАЯ ЗАМЕТКА"

**Не-каноничный, Mia solo.** Дневниковая зарисовка — Мия записывает себе voice memo, неважно когда и где. Без сюжета, без локации, чистая воспоминалка. Можно повесить в любую точку timeline или вообще вне канона.

- Звук: почти a cappella — голос + минимум пианино, без барабанов вообще
- Длина: ~2:30
- Минимум продакшена → быстрее всего собрать
- Тест MIYA_v1 в **самом обнажённом виде** (без подложки маскирующей дефекты)
- Можно использовать как **тизер артиста** в соцсетях — короткий, интимный, "ага вот её голос"
