## Критерии финального выбора

При выборе имени проверить:

- [ ] Свободно в Spotify (нет другого артиста с таким же именем)
- [ ] Свободно в Apple Music
- [ ] Доступен домен (.com / .ru / .me)
- [ ] Свободно в Instagram / TikTok / VK / Telegram
- [ ] Читается на латинице (для международных площадок)
- [ ] Не пересекается с известным брендом / товарным знаком
- [ ] Звучит хорошо, когда произносишь вслух
